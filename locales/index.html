Convention reference
  • Locale code = language-REGION. Language lowercase (ISO 639-1), region UPPERCASE (ISO 3166-1), hyphen delimiter.
  • Multi-script languages use a titlecase 4-letter ISO 15924 script subtag instead of region, e.g. sr-Latn.
  • Exactly one default locale per language.
  • Marketplace prefix is derived: default locale → language subtag alone; non-default → full locale code. English default (US) → None (site root).
  • Always the modern code: he not iw, id not in. Language ≠ TLD: cs not cz, el not gr, uk not ua.
